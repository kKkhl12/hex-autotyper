Auto-Typer Script Setup Guide

Follow these simple steps to get the auto-typer script running on your computer.

Step 1: Download the Script

Click the button below to download the Python script file.

Download Legit_auto_typer.py

Step 2: Download & Install Python

If you don't have Python, you'll need to install it. You can get the latest version for your system from the official website.

Download Python from Python.org

The most important part of the installation is to check the box that says "Add Python to PATH". This allows you to run Python from any folder on your computer. The video below clearly demonstrates this process for Windows:

Step 3: Install Required Libraries (Modules)

This script needs a specific library called keyboard to work. You can install it using Python's package installer, `pip`.

Good News! The other libraries used, tkinter and ttk, are part of Python's standard library, so you do not need to install them separately. They are included when you install Python.

  1. Open your command line tool.
    • On Windows, press the Windows Key, type cmd or PowerShell, and press Enter.
    • On Mac/Linux, open the Terminal application.
  2. Install the `keyboard` library.

    Type the following command into your terminal and press Enter:

    pip install keyboard

    You should see some text as it downloads and installs the package.

Step 4: Run the Auto-Typer Script

Once everything is installed, you just need to run the script.

  1. Navigate to your Downloads folder. In the same terminal window you opened, type the following command and press Enter. 
    cd Downloads
  2. Run the script. Now, use Python to execute the file. Type this command and press Enter: 
    python Legit_auto_typer.py

The script's window should now appear, and you can start using it!
